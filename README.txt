Contrato correspondiente a la entrega 2 de la microcredencial [“Tecnologías Blockchain, Contratos Inteligentes, Aplicaciones Descentralizadas y la Web 3.0” de la Universida de Oviedo](https://www.uniovi.es/en/estudia/titulospropios/microcredenciales/oferta/-/asset_publisher/NCtkAT4jPSMb/content/id/10937457)

## Cómo funciona

Primero es necesario desplegar este contrato en la red Blockchain mediante [Remix IDE](https://remix.ethereum.org/) utilizando el script deploy_tih_ethers.ts
Después, se debe iniciar la [aplicación](https://github.com/alfonsodou/app-listaespera) para poder interactuar con el contrato.
